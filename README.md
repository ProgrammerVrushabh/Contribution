# Vocational Training

This repository contains a simple calculator application built with Python and PyQt5.

## Project contents
- Calculator/Calc.py: main calculator application logic
- Calculator/Calculator.ui: Qt Designer UI definition

## Requirements
Install the required dependency:

```bash
pip install PyQt5
```

## Run the application
From the repository root, run:

```bash
python Calculator/Calc.py
```

## Notes
The calculator supports basic arithmetic operations and a percentage button.
